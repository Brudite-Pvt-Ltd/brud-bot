
# Discord Bot 
Automated programs that look and act like users and automatically respond to events and commands on Discord are called bots 




## Installation

Prerequisites - 

* python 3.11.4  
* discord server  
* discord bot

Steps -  

Clone the project.  
Install discord.py  
Update the BOT_TOKEN


## Commands 
All Commands starts with '.'  
* .hi or .hello
* .help
* .joke
